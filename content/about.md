This site is built with [Hugo](https://gohugo.io/) using the [XMin theme](https://github.com/yihui/hugo-xmin) designed by [Yihui Xie](https://yihui.org) — an extremely minimal theme, about 140 lines of HTML and CSS in total, built with the philosophy of keeping things simple without sacrificing readability.

The theme's clean design and zero-JavaScript approach made it the perfect foundation for this site.

If you're looking to build your own Hugo site, XMin is a great place to start.
